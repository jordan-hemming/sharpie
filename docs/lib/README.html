    Unofficial Curses Native Libraries

    This directory contains unofficial builds of the native libraries supported by Sharpie. All libraries are built using GitHub workflows.

    These libraries are provided AS IS with NO WARRANTY whatsoever from my side.

    NCurses

    NCurses libraries are built for Linux, MacOs and Windows using this workflow. You can check it out to see the used compilation settings. The original source repository is located here

    PDCursesMod

    PDCursesMod libraries are built for Linux, MacOs and Windows using this workflow.

    The libpdcursesmod-vt-arm64.dylib library has been built on a Mac M1 locally. The pdcursesmod-vt-arm64.dll, pdcursesmod-wincon-arm64.dll and pdcursesmod-wingui-arm64.dll libraries have been built on a Windows 11 ARM locally.

    The original source repository is located here
